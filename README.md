# USE PYTHON 3x

# INSTALLATION GUIDE:

## Install virtual environment: 
`virtualenv -p python3 envname` 
## Activate the environment: 
`source envname/bin/activate`
## Install the requirements using requirements.txt file: 
`pip install -r requirements.txt`

## Run the file: 
`python detect.py`
